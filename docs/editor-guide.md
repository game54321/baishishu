# 百世书 - 资源编辑器使用文档

## 启动编辑器

```bash
cd editor
npm install
npm run dev
```

编辑器运行在 `http://localhost:5173`，Express 后端运行在 `http://localhost:3001`。

---

## 界面概览

编辑器分为三个区域：

- **左侧面板** — 节点类型 / 功法 / 章节管理
- **中央画布** — 关卡地图编辑（拖拽节点、连线）
- **右侧属性面板** — 选中节点后配置消耗/产出卡牌

顶部工具栏：
- **保存并导出** — 保存所有修改并导出 Godot 格式数据
- **连线** — 切换连线模式（点击进入后依次点击两个节点建立连线）

---

## 一、功法管理

### 新增功法
1. 点击左侧 **功法** 标签
2. 点击 **新增** 按钮
3. 填写：
   - **ID** — 唯一标识（如 `wild-dog-fist`）
   - **名称** — 显示名称（如 野狗拳）
   - **描述** — 功法效果说明
   - **颜色** — 功法主题色
   - **插图** — 上传 `.png/.jpg/.webp`，展示在卡牌和背包中
   - **击中特效** — 上传战斗中使用时敌人身上的特效图
   - **基础伤害** — 每次使用造成的伤害
   - **获取经验** — 每次使用获得的熟练度

### 编辑 / 删除功法
- 点击列表中功法条目进入编辑
- 点击红色垃圾桶图标删除

### 清理未使用插图
- 标题栏点击 **清理未用插图** 按钮，删除 `assets/功法插图/` 中未被任何功法引用的图片文件。

---

## 二、章节管理

### 创建章节
1. 点击左侧 **章节** 标签
2. 输入章节名，点击添加
3. 选中章节即可编辑该章的地图

### 切换章节
- 点击章节列表切换，编辑器会自动保存当前章节的修改

---

## 三、地图编辑

### 添加节点
1. 点击左侧 **节点** 标签
2. 从节点类型列表中点击任意类型（战/精/休/店/武等）
3. 新节点会出现在画布中央

### 移动 / 选中节点
- 拖拽节点到任意位置
- 点击节点打开右侧属性面板

### 连线
1. 点击顶部 **连线** 按钮进入连线模式
2. 依次点击两个节点建立连线
3. 再次点击 **退出连线** 退出

### 删除连线
- 选中节点 → 右侧属性面板底部 → 点击入边/出边的 **删除** 按钮

### 节点属性配置
选中节点后可配置：
- **节点类型** — 切换类型
- **武馆名称**（dojo 节点）/ **工作名称**（tavern 节点）
- **消耗卡牌** — 玩家进入需要消耗的资源
- **产出卡牌** — 完成后获得的资源
  - 产出类型选"功法"时可选择具体功法和经验值

---

## 四、节点类型管理

- 左侧 **节点** 标签下方点击 **新增类型**
- 可配置：ID、名称、类别、颜色、出现权重、楼层范围、每图上限

---

## 五、数据存储

编辑器数据保存在 `editor/data/` 目录，导出到游戏数据目录 `data/`：

| 文件 | 说明 |
|------|------|
| `data/card_types.json` | 卡牌类型定义 + 功法列表 |
| `data/node_types.json` | 节点类型定义 |
| `data/chapters.json` | 章节 + 地图数据 |
| `data/maps/*.json` | 历史地图文件（编辑器用） |

---

## 六、静态部署

```bash
cd editor
npm run build
```

构建产物在 `editor/dist/`，配合 Express 后端可直接部署为静态站点。

Vite 配置中已设置生产环境 `base` 为 `/baishishu/editor/`，如需修改请编辑 `vite.config.ts`。
