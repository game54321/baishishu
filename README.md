## 简介

**百世书**是一款以修仙风格的卡牌游戏，玩家可以转世100次，每次转世可以继承5张卡牌(游戏刚开始策划, 还没确认最终继承的次数，考虑后面在游戏中增加继承次数的道具) ,

游戏中所有资源都抽象成为卡牌, 玩家修为是卡牌，玩家寿命也是卡牌例如有60年寿命就是：一年时间x60， 游戏流程是经过类似杀戮尖塔的网状路线，对自身拥有的卡牌进行增值， 为这一世或下一世铺路

游戏内置react + vite编写的web资源管理器(/editor)，可以管理游戏中所有美术资源和游戏数值配置，例如所有功法卡牌都是由资源管理器动态管理的，方便大家自定义

本人之前全栈开发，第一次接触游戏开发，欢迎大家提issue和pr

这游戏是6月8号突发奇想立项，项目现在还属于摸索期，需要先用ai代码屎山为游戏探索出核心机制玩法, 暂时无法保证代码质量。

### 环境要求
- [Godot 4.6](https://godotengine.org/download) 或更高版本
- Node.js 22+ (仅编辑器需要)

## 技术栈
- **游戏引擎:** Godot 4.6 (GDScript)
- **资源编辑器:** React 18 + Vite + Express.js
- **数据:** JSON (纯配置文件驱动)
- **UI 设计:** 黑白扁平化 + 暖色水墨卷轴
